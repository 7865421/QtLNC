# QtLNC
Using QT5, connect to the Baoyuan CNC system, obtain the corresponding parameters of the CNC system, and implement permission control, including Modbus protocol.

1. Use qt5 32-bit mingw32 compiler
2. When the program is running, only one instance can run on the local machine;
3. Use SQilte to operate the database
4. Includes FTP client to enable file upload and download between servers
5. Permission control, divided into 0-7, can achieve card swiping login;
6. Data collection from external Modbus devices
7. HTTP protocol, retrieve JSON data through an interface and analyze it.
8. The report can be exported to Excel, with some exported as CSV
